# Smart Beauty Task

Smart Beauty — Frontend / Full-Stack Stajyer Vaka Çalışması.

Kullanıcının cilt tipini (Kuru / Yağlı / Karma) seçtiği, 3 sn'lik bir "AI analizi" simülasyonu sonrasında seçimine uygun cilt bakım ürünlerinin listelendiği ve sepete eklenebildiği mini bir e-ticaret akışı.

> Kurulum, çalıştırma ve deploy bilgileri proje tamamlandığında bu dosyaya eklenecektir.

## Klasör Yapısı

```
SmartBeautyTask/
├── backend/    # Express + TypeScript API (3 sn delay'li ürün endpoint'i)
└── frontend/   # Next.js (App Router) + TypeScript + Tailwind + Zustand
```
